# 🎮 Memory Flip Card Game

A fun and interactive memory card game built with HTML, CSS, and JavaScript. Match pairs of emoji cards to complete the game!

## 🎯 Features

- **Emoji Cards**: Beautiful animal emoji images behind each card
- **Score Tracking**: Earn points for each successful match
- **Move Counter**: Track how many moves you make
- **Timer**: See how long it takes to complete the game
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Animations**: Beautiful card flip animations and effects
- **Game Over Screen**: Celebrate your victory with final statistics

## 🎮 How to Play

1. **Start the Game**: Click on any card to begin
2. **Find Matches**: Click on two cards to reveal their emoji images
3. **Match Pairs**: Find two cards with the same emoji to create a match
4. **Complete the Game**: Match all 8 pairs to win!

## 🎨 Game Features

- **16 Cards**: 8 pairs of matching emoji cards
- **Random Shuffle**: Cards are shuffled each new game
- **Visual Feedback**: Matched cards turn green and stay revealed
- **Score System**: +10 points for each successful match
- **Timer**: Tracks your completion time
- **Move Counter**: Counts every pair of cards you flip

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. Click "🔄 New Game" to start a fresh game
3. Click on cards to flip them and find matches
4. Complete all pairs to see your final score!

## 📱 Responsive Design

The game adapts to different screen sizes:
- **Desktop**: 4x4 grid layout
- **Tablet**: 3x3 grid layout  
- **Mobile**: 2x2 grid layout

## 🎯 Game Statistics

- **Score**: Points earned from successful matches
- **Moves**: Number of card pairs flipped
- **Time**: Total time to complete the game

## 🎉 Victory Screen

When you complete all matches, you'll see:
- Final completion time
- Total moves made
- Final score achieved
- Option to play again

## 🛠️ Technical Details

- **CSS3**: Modern styling with gradients and animations
- **JavaScript ES6+**: Object-oriented game logic
