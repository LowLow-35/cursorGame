# 🎮 Memory Flip Card Game

A fun and interactive memory card game built with HTML, CSS, and JavaScript. Match pairs of emoji cards to complete the game!

## 🎯 Features

- **Emoji Cards**: Each card features a different animal emoji
- **Smooth Animations**: Beautiful flip animations and hover effects
- **Game Statistics**: Track your moves and time
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Game Over Screen**: Celebrate your victory with final stats
- **Reset Functionality**: Start a new game anytime

## 🎮 How to Play

1. **Start the Game**: Click on any card to begin
2. **Find Matches**: Click on two cards to reveal their emojis
3. **Match Pairs**: Find two cards with the same emoji to create a match
4. **Complete the Game**: Match all 8 pairs to win!

## 🎨 Game Features

- **4x4 Grid**: 16 cards total (8 pairs)
- **Random Emojis**: Each game uses different emoji combinations
- **Move Counter**: Track how many moves you make
- **Timer**: See how long it takes to complete the game
- **Visual Feedback**: Matched cards turn green and stay flipped
- **Smooth Transitions**: Cards flip back if no match is found

## 🚀 How to Run

1. Download all files to a folder
2. Open `index.html` in your web browser
3. Start playing immediately!

## 📁 Files Included

- `index.html` - Main HTML structure
- `styles.css` - Beautiful styling and animations
- `script.js` - Game logic and functionality
- `README.md` - This file

## 🎯 Game Rules

- Click on any card to flip it
- You can only flip two cards at a time
- If the cards match, they stay flipped and turn green
- If the cards don't match, they flip back after 1 second
- The game ends when all 8 pairs are matched
- Use the "Reset Game" button to start over anytime
